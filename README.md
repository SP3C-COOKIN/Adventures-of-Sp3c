# Adventures of SP3C

A personal journal of a fictional multiversal traveler exploring different worlds and realities.  
The project presents his experiences through an interactive web interface, blending storytelling with visual design.

---

## Current State

- Glassmorphism UI with floating cards for the title and introduction  
- Cosmic background to establish the visual tone  
- A canvas-based cosmic simulation representing different universes  

Each sphere represents a world or journal explored by SP3C, moving in structured yet dynamic patterns.

The motion system is currently evolving:
- Initially used velocity-based bouncing (ping-pong style)
- Now transitioning to a sine wave orbital system for smoother, more natural movement

---

## Tech Used

- HTML, CSS, JavaScript (Vanilla)
- Canvas API
